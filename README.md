# Задание 1

### Текст задачи

На основе данного списка строк получить новый, где к каждой строке в конце
дописан заданный символ.

### Алгоритм решения

1)Запрашиваем у пользователя строку (через пробел)
2)Вводим строку
3)Разделяем строку по пробеам и получаем массив
4)Получаем список из массива
5)Запрашиваем символ, который будем вставлять в конец каждой строки
6)Вводим символ
7)Вызываем  функция добавления символа в конец строки
7.1)list.map принимает два параматера - функция преобразования (функция добавления символа в конец строки) которая применяется к каждой строке и исходный список, в результате получаем новый измененный список 
8)Выводим список по строчке

## Задание 2

### Текст задачи

Список содержит строки. Найти самую короткую строку.

### Алгоритм решения

1)Запрашиваем у пользователя строку (через пробел)
2)Вводим строку
3)Разделяем строку по пробеам и получаем массив
4)Получаем список из массива
5)вызываем функцию поиска самой короткой строки
5.1)list.fold для поэлементного обхода списка, для каждого элемента применяет функцию (накопления), сравниваем длинну строк и выявлем самую короткую строку
6)Выводим самую короткую строку
