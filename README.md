# Задание 1

### Текст задачи

На основе данного списка строк получить новый, где к каждой строке в конце
дописан заданный символ.

### Алгоритм решения

1)Запрашиваем у пользователя строку (через пробел)
2)Вводим строку 
3)Получаем массив из строк
4)Запрашиваем символ, который будем вставлять в конец каждой строки
5)Вводим символ
6)Вызываем  функция добавления символа в конец строки
6.1)Склеить строку с символом
7)Вывести список
